1. Install PHP
see http://www.hell-desk.com/arduino-yun-thermostat-34-configuring-arduino-yun/
> suivre les instructions pour installer le paquet php

Webserber UHTTPD
> edit /etc/config/uhttpd
> uncomment the entry relative to the ph interpreter

> modifiy the number of concurrent requests to 5

        # Maximum number of concurrent requests.
        # If this number is exceeded, further requests are
        # queued until the number of running requests drops
        # below the limit again.
        option max_requests 5

        # Certificate and private key for HTTPS.
        # If no listen_https addresses are given,
        # the key options are ignored.
        option cert             /etc/uhttpd.crt
        option key              /etc/uhttpd.key

        # CGI url prefix, will be searched in docroot.
        # Default is /cgi-bin
        option cgi_prefix       /cgi-bin

        # List of extension->interpreter mappings.
        # Files with an associated interpreter can
        # be called outside of the CGI prefix and do
        # not need to be executable.
        list interpreter        ".php=/usr/bin/php-cgi"
#       list interpreter        ".cgi=/usr/bin/perl"


> save and restart the Webserver
/etc/init.d/uhttpd restart
